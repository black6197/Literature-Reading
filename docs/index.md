# Literature Reading 📚

欢迎来到我的文献阅读笔记库！

## 📖 关于本站

这是我整理的学术文献阅读笔记和资源汇总，主要聚焦于：

- **足球分析** - 使用数据科学和机器学习分析足球比赛
- **强化学习** - 在序列决策问题中的应用
- **深度学习** - Transformer 等先进模型
- **运动分析** - 体育数据分析方法论

## 🎯 研究方向

目前主要关注：

1. **Expected Contribution Value (xCV)** - 球员贡献价值评估
2. **强化学习在足球中的应用** - 使用 TD Learning 评估球员动作
3. **Transformer 架构** - 序列建模和注意力机制
4. **360度追踪数据分析** - 结合位置数据的上下文分析
