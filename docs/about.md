# 关于

## 👤 关于我

我是一名博士研究生，主要研究方向为：

- **足球数据分析** - 使用机器学习评估球员表现
- **强化学习** - 序列决策和价值评估
- **深度学习** - Transformer 等模型架构

## 🔬 当前研究

正在开发 **xCV (Expected Contribution Value)** 框架，这是一个基于强化学习的足球球员动作评估系统，主要创新点包括：

1. **链式分析** - 分析完整的控球序列而非单个事件
2. **学习型奖励** - 使用神经网络学习终端价值而非手工设计
3. **上下文感知** - 结合 360 度追踪数据考虑场上情况

目标是在 **KDD 2026** 发表研究成果。
